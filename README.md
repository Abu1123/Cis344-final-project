Real Estate Agency Portal

This project is a web based Real Estate Agency Portal built using PHP, MySQL, HTML, and CSS. It allows agents, buyers, and renters to interact with property listings in an organized way.

Features
- View all properties with details and images
- Search properties by city and price
- Add and delete properties (agents only)
- Buy or rent properties
- Save properties to favorites
- Submit inquiries
- User registration and login system
- Role based access (agent, buyer and renter)
  
How to Run:
1. Start Apache and MySQL
2. Import the database into phpMyAdmin
3. Place the project folder in htdocs
Open: http://localhost/your-folder-name/

Notes:
- Database name: real_estate_portal_db
- Make sure stored procedures, trigger and view are created
- Images are stored in the uploads folder and it is automatically created
